1) Para o plugin dockerfile-maven-plugin funcionar é necessário expor daemon na porta 2375, mas não é recomendavel

2) Criar o arquivo C:\Users\<usuario>\.m2\settings.xml
<settings xmlns="http://maven.apache.org/SETTINGS/1.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
  xsi:schemaLocation="http://maven.apache.org/SETTINGS/1.0.0 https://maven.apache.org/xsd/settings-1.0.0.xsd">
    <servers>
        <server>
            <id>docker.io</id>
            <username>login</username>
            <password>senha</password>
        </server>
    </servers>
</settings>


SET JAVA_HOME=C:\Program Files\Java\jdk-16.0.2
mvnw clean package dockerfile:push
